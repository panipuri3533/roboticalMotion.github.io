This is the second version of my first creation. this version is fully automated from where it turns on untill when the turret on the top turn on. I changed how the touch sensor works and the way it is placed so that when the robot gets bumped into something, it turns on the turret! Even though this is a simple robot structure, keep in mind this was one of my starting creations so I had to get used to it! As you keep digging deep, the creations get complex and complex!

This robot is almost the same as the one before but instead of 1 set of tracks, it uses 2 so that it is balanced on each side. Also, the code was changed so that the gyro sensor makes the robot help go in a straight line so that every thing is accurately meassured and acted upon. The robot is also programed to use the ultra sonic sensor so that it can determine when to stop if an object is blank amounto of units away from the robot.
